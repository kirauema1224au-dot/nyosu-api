これから Node.js + MySQL を使う API プロジェクト「nyosu-api」の開発環境を、このPC上に整えてください。

\## やりたいこと



1\. Node.js のバージョンを揃える（nvm + .nvmrc）

2\. npm パッケージをインストールする

3\. MySQL に開発用データベースとユーザーを作成する

4\. .env を作成する

5\. サーバーを起動して動作確認する



\## 前提



\- OS: Windows 10/11 + WSL2(Ubuntu) または MacOS を想定

\- このフォルダに以下のファイルがある前提：

&nbsp; - package.json

&nbsp; - .nvmrc

&nbsp; - .env.example

&nbsp; - db/schema.sql
